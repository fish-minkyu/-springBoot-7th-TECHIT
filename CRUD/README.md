# CRUD - MyBatis 프로젝트

- 2024.01.04 `7주차`

ArrayList를 DB삼아 CRUD를 수업, thymeleaf를 통해 화면 구현을 하였으며  
MyBatis(SQLite 연결)으로 CRUD을 구현하였다.

## 스택
- Spring Boot
- Spring Web
- Thymeleaf
- Lombok
- MyBatis Framework
- SQLite
